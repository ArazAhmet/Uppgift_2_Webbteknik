Layout

  • Förklara vad CSS-egenskaperna justify-content respektive align-items har för effekt när man använder flexbox för layout.

    justify-content och align-items används för att styra placeringen av element i en flexbox. justify-content justerar elementens horisontella position om flex-riktningen inte är "column", i vilket fall den istället justerar vertikalt. align-items justerar elementens vertikala position, men om flex-riktningen är "column", fungerar den horisontellt istället.

  • Vad är en media query och vad används den till?

    Media query i CSS är ett verktyg som möjliggör anpassning av webbsidor för olika enheter och skärmstorlekar. Med hjälp av media queries kan du bygga responsiva sidor som dynamiskt justeras beroende på bland annat skärmens storlek.
